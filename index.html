Recent news

Version 2.1.1

As there is still no version by the team here, I released a new version.

This version is stable. /rm limits and /rm list does not work yet. You can set limits when you create the limits.yml manually (example below; use the YAML format; -1 means often infinite; If you dont know how: Let it).

limits.yml (name means the name of the template; world means the world; etc.)

  • global.global - Global limit
  • name.global - Global template limit
  • global.world.world - Global world limit
  • name.world.world - Template world limit
  • global.parentregion.parentregion - Global limit by parentregion (NOT priority)
  • name.parentregion.parentregion - Template limit by parentregion
